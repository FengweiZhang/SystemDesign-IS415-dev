# 内核模块相关功能清单

## 完成功能

`prm_hook`

- [x] 找到系统调用表位置

- [x] 对系统调用进行hook

`prm_netlink`

- [x] 基础数据通信

- [x] 包含信息类型的数据通信

- [x] 进程间数据通信（使用信号量与全局变量）


`user_netlink`

- [x] 与内核态建立连接

- [x] 与内核态基础通信，包含信息类型的通信

- [x] 重新建立/断开与内核态的连接
